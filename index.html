    0xAcid：结合复合资产+真实收益的质押协议
    即将在 3 月到来的以太坊上海升级预计将为 LSD 赛道带来更多的热度，许多新的 LSD 项目正在抓紧机会加入战场。 0xAcid 是一个旨在最大化 LSD 资产回报的管理协议，力求为用户带来远高于普通 LSD 资产的回报，将于 Arbitrum 和以太坊主网上线。对于一个 DeFi 协议来说，Token 设计至关重要。目前只有 Uniswap 这样的顶级 DeFi 协议可以不为自己的 Token 寻找价值，却仍能保持价格。其他的 DeFi 协议都在寻求为其 Token 持有者带来真实收益（Real Yield）。 0xAcid 产生的所有收入，都将分配给其主要 Token ACID 的持有者。据官方介绍，0xAcid 可以为用户提供 stETH、cbETH 等普通 LSD 资产 2-6 倍的收益，这样的收益将如何实现？ 复合资产管理策略 0xAcid 通过一套组合策略产生持续收入： -管理积累大量 LSD 资产如 stETH、rETH、frxETH 获得以太坊节点收入； -将管理的资产投入 DeFi 获取收益 90% 投资于较稳定回报低风险项目，10% 投资于稍高风险的较稳定回报项目如 Curve、Balancer 等。在上线初始阶段，0xAcid 采用的主要 DeFi 策略是将 30% 的资产转换为 frxETH 放入 Frax，70% 存入 Aura Pool 的 wstETH 3pool； -0xAcid 协议含有协议流动性控制机制，可获取大部分的 Token swap 收益，0xAcid 将于 Sushiswap 添加 LP，持有大部分的 ACID LP Token，并设置 1% 的 swap 手续费； -0xAcid 具有借贷服务，可让用户质押 ACID 并借出 wstETH，收取借贷利息，并 100% 分配给 ACID 质押者； -底层支撑资产 ETH 升值带来的收益。 对于 Token 持有者来说，还可质押锁仓 ACID，获得高额 esACID 奖励，esACID 可归属转换为 ACID 出售或继续质押。 结合多种真实收益的 Token 机制设计 0xAcid 协议有两种主要 Token，ACID 与 esACID。ACID 总量无上限，初始供应为 7500 枚，Public sale 5000 枚，团队保留 750 枚并质押锁定 1 年，用于提供初始流动性 250 枚，DAO 国库保留 1000 枚进行质押但不锁定，另有 500 枚用于空投和宣传。   ACID 将主要在协议上线初期由 esACID 的排放来快速生成，esACID 的排放将随时间推移而逐渐下降。在上线第一周，每天排放 2000 枚 esACID，第二周每天 1500 枚，第三周 1000 枚，第三周之后 500 枚，具体排放数量可由 DAO 投票修改。 ACID 持有者的真实收益 0xACID 协议产生的所有利润都将以 ETH 或 stETH 的形式分配给质押和锁定 ACID 的持有者。此外，质押 ACID 的用户还可获得 esACID 奖励。质押时间越长，能获得的奖励也就越多，封顶锁定时间为 1 年。 esACID 有两种作用，第一是参与协议的治理，第二是可将 esACID 归属转换为 ACID。   为了将 esACID 转换为 ACID，需要先质押 200% 的 ACID。一旦满足了质押要求，转换过程就会开始，esACID 将在 30 天内线性转换成 ACID。用债券带来流动性持有者可锁定其持有的 ACID，将它们债券化（Bonding），用户购买折价的 ACID 债券后，可获得逐渐释放的 ACID Token。此过程可为 0xAcid 带来流动性和更多的 LSD 资产。 质押借贷 用户可以使用 ACID 作为抵押品，通过质押 ACID 借入 wstETH，在此过程中 0xACID 协议将收取利息，并将利息分配给 ACID 持有者。
