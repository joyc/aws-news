    原文标题：《Game 2.0》 原文作者：guiltygyoza 原文编译：Kxp，BlockBeats 今天我们正处于游戏 1.0 阶段，但随着可验证计算技术的发展，我们将迎来游戏 2.0 时代。 电子游戏经历了多轮演变，从早期的雅达利游戏、街机游戏、家庭游戏，到多人联网游戏和手机游戏——这一系列演变塑造出了电子游戏的各种属性——丰富的互动性、社交性、竞争与匹配机制、沉浸感、非线性叙事、按需获取多巴胺驱动的游戏循环——所有这些都归功于内存容量、计算能力、视听保真度、网络速度以及覆盖范围的指数级增长。 区块链技术对电子游戏产生了与以往完全不同的影响。如果说手机的出现让游戏有了休闲性，那么区块链技术则将真正赋予游戏以重要性。区块链是一个「强劲」的计算平台，它在管理状态差异上的规则可以抵御各种类型的国家级攻击。这意味着，「休闲类链上游戏」会成为一个矛盾修饰语。不过话说回来，区块链的引入究竟将如何改变电子游戏？ 我们至少可以从两个趋势中探寻到蛛丝马迹：电子游戏中用户生成内容的增加，以及链上 DeFi 协议的兴起——这两个趋势向我们展示了人类欲望的一些基本特征：渴望以创造者的身份去积极地创造和表达，而不是一味地被动消费。在创作内容时，无论公开编辑页面或许可协议有着怎样的限制，玩家都可以体验一把成为游戏开发者的感觉。 在无需许可的智能合约完成部署之后，程序员便可以设计出公开透明的金融产品，任何能够接触到底层区块链或 Rollup 的人都将能够直接获取到这些产品——这在几年之前还是无法想象的事情。 最近的统计数据显示，在美国，游戏玩家与游戏行业员工之间存在着约 1000 倍的数量差。借用 Web 1.0 与 Web 2.0 的概念来看的话，我们今天仍主要处于游戏 1.0 阶段。在我们的印象中，游戏玩家的角色也仍然是游戏逻辑与内容的消费者而非创造者。 因此，有人预测说区块链将引领我们进入游戏 2.0 时代。随着所有权在强不可变类执行平台上的成功实现，这些平台的计算能力因可验证计算的不断创新而得以扩展（证明系统、证明递归和聚合、Rollup 层架构、序列器去中心化、硬件加速等），再加上其编程范式正努力实现无需许可的可组合性和互操作性，我们正走向这样一个未来：全世界的人都能直接参与到核心游戏系统的创造与升级——不仅仅能对游戏物品、模式和地图进行定制，还可以用模块化、可组合的方式对游戏角色、AI 系统与物理系统做出设计，并且能借助互操作性真正拥有他们在不同平台或生态系统中所创建物品的所有权。 原文链接 律动 BlockBeats 提醒，根据银保监会等五部门于 2018 年 8 月发布《关于防范以「虚拟货币」「区块链」名义进行非法集资的风险提示》的文件，请广大公众理性看待区块链，不要盲目相信天花乱坠的承诺，树立正确的货币观念和投资理念，切实提高风险意识；对发现的违法犯罪线索，可积极向有关部门举报反映。
